У тебя есть текстовый файл, в котором есть значения от 0 до 20 в произвольном порядке
(перемешаны). Значения указаны через запятую.
1. Прочитай значения из файла, отфильтруй значения по возрастанию и выведи в
консоль.
2. Прочитай значения из файла, отфильтруй значения по убыванию и выведи в
консоль.

МЕТОД readFile(): читает значения из файла, создает массив чисел
СЧИТЫВЕМ файл посимвольно
ПОКА символы не закончились
	ЗАПИСЫВАЕМ символ
	ИНАЧЕ выходим из цикла и возвращаем строку
ДЕЛИМ строку по запятой и получаем массив строк
ПРИВОДИМ массив строк к массиву чисел

МЕТОД upArrangement: упорядочивает значение от меньшего к большему и выводит в консоль

МЕТОД downArrangement: упорядочивает значения от большего с меньшему и выводит в консоль
